Squiral

The assignment was to make modrern art with polygons. I used the program Snap! to create the project. Drawing App

Drawing App

The assignment was to creat a drawing app for the congressional app challenge. I used the program Snap! to create the project. To operate the drawing app, first you press "d" to exit the loading screen. Next, you would click the draw shape or draw polygon tools to create a shape. You can use shape to draw. I learnt how to find the center of a regular polygon and I had a challenge with the math to find the center of a regular polygon.
